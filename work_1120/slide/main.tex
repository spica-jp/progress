    \section{k分割交差検証（比率を考慮）}
\end{frame}

\begin{frame}{手順}
    \begin{enumerate}
        \item クラスごとに画像データとラベルをそれぞれ分ける
        \item クラスごとに画像データとラベルをk個に分ける
        \item 以下の操作をk回繰り返す
        \begin{enumerate}
            \item それぞれのクラスにおいてk個に分けた画像データとラベルデータを検証データ用に一つ選ぶ
            \item 検証データ以外を訓練データとしてまとめる
            \item 機械学習を行う
        \end{enumerate} 
        \item すべての学習終了後、ロスと正解率の平均を計算する
    \end{enumerate}
\end{frame}
